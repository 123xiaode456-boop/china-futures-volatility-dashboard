# 国内商品期货波动率网页 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a static website that displays daily domestic commodity futures volatility and can be deployed to GitHub Pages with scheduled data updates.

**Architecture:** Python scripts compute volatility metrics and emit `site/data/volatility.json`. A static HTML/CSS/JavaScript frontend reads that JSON, renders summary cards and a sortable/filterable table, and remains deployable without a server. GitHub Actions runs tests, refreshes data, and deploys the `site` folder to Pages.

**Tech Stack:** Python standard library plus optional `akshare`, static HTML/CSS/vanilla JavaScript, Node.js built-in test runner, GitHub Actions, GitHub Pages.
