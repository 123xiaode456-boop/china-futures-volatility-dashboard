# 商品历史波动率搜索图表 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the market-wide volatility table with a searchable single-contract historical candlestick and volatility chart.

**Architecture:** Python generates a historical data package with selector metadata and per-contract daily series. The static frontend reads the package, filters one selected contract by date range, and draws price candlesticks plus 20/60-day volatility lines on Canvas. Deployment remains GitHub Pages with the same repository and public URL.

**Tech Stack:** Python standard library + AKShare, vanilla HTML/CSS/JavaScript, Canvas 2D, Node built-in test runner, GitHub Pages.
